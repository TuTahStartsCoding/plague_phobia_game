# Plague Phobia Game - เกม Horror Phobia แนว Pixel Top-Down

เกมสยองขวัญแนว pixel art ที่คุณจะต้องออกเดินทางหายาสมุนไพรเพื่อรักษาผู้ป่วยจากโรคระบาด

## เนื้อเรื่อง

ในยุคที่โรคอหิวาตกโรคและวัณโรคแพร่ระบาดไปทั่วหมู่บ้าน คุณคือหมอพื้นบ้านคนสุดท้ายที่ต้องออกเดินทางหายาสมุนไพรในป่าเพื่อรักษาผู้คนที่กำลังต่อสู้กับโรคร้าย แต่ป่าแห่งนี้มีอันตรายรออยู่มากมาย...

## วิธีเล่น

### การควบคุม
- **WASD** หรือ **ลูกศร**: เคลื่อนที่
- **E**: โต้ตอบ/เก็บของ/รักษาผู้ป่วย
- **Shift**: วิ่ง (ใช้พลังจิตใจ)

### เกมลูป

#### ลูปที่ 1: ภารกิจสำเร็จ
1. หายาสมุนไพรในป่า (ต้องการ 3 ชนิดต่อผู้ป่วย 1 คน)
2. นำยามารักษาผู้ป่วยภายในเวลาที่กำหนด
3. ช่วยผู้ป่วยรอด → ดำเนินต่อไปจนครบจำนวนที่กำหนดในแต่ละวัน
4. ครบจำนวนวัน → **ค้นพบยารักษาโรคระบาด (ชนะเกม)**

#### ลูปที่ 2: ผู้ป่วยเสียชีวิต
1. หายาสมุนไพรในป่า
2. รักษาผู้ป่วยไม่ทัน → ผู้ป่วยเสียชีวิต
3. จิตใจถูกกระทบ เกิดภาพหลอน
4. รีบเข้าวัดเพื่อสวดมนต์รักษาจิตใจ
5. หลบเลี่ยงภาพหลอนระหว่างการรักษา
6. จิตใจดีขึ้น → กลับไปหายาสมุนไพรต่อ

**คำเตือน**: หากผู้ป่วยเสียชีวิตมากเกินไป จะเกิดภาพหลอนจนควบคุมไม่ได้ → **Game Over**

## ระบบเกม

### 1. ระบบสมุนไพร
- **ชนิดสมุนไพร**: สะระแหน่, ขิง, ขมิ้น, กระเทียม
- ต้องการ **3 ชนิดต่อผู้ป่วย 1 คน**
- สมุนไพรจะ respawn หลังจาก 60 วินาที

### 2. ระบบสุขภาพจิต
- **Mental Health**: สุขภาพจิตของคุณ (0-100%)
- ลดลงเมื่อ:
  - ผู้ป่วยเสียชีวิต (-15%)
  - โดนวิญญาณ (-20%)
  - อยู่ในพื้นที่อันตราย (-5%/วินาที)
- เพิ่มขึ้นเมื่อสวดมนต์ที่วัด

### 3. ระบบภาพหลอน
- **Hallucination Level**: ระดับภาพหลอน (0-100%)
- เพิ่มขึ้นเมื่อผู้ป่วยเสียชีวิต (+20%)
- เมื่อถึง 50% จะถูกบังคับเข้าวัด
- เมื่อถึง 100% → **Game Over**

### 4. ระบบโรค
- ติดเชื้อจาก:
  - น้ำเสีย/ของเสียที่ปนเปื้อน
  - โดนวิญญาณ
- มีเวลา 30 วินาทีในการรักษาก่อนจะเสียเลือด
- รักษาได้ที่วัด

### 5. อันตรายในเกม
- **วิญญาณ**: หลอกหลอนในป่า โจมตีทำให้เสียสุขภาพจิตและต้องเข้าวัด
- **น้ำเสีย**: พื้นที่ปนเปื้อน ทำให้ติดโรคและเสียเลือด
- **เวลา**: ต้องรักษาผู้ป่วยให้ทันก่อนหมดเวลา

## โซนในเกม

### 1. หมู่บ้าน (Village)
- จุดเริ่มต้นเกม
- พบผู้ป่วยที่ต้องการการรักษา
- ปลอดภัย ไม่มีอันตราย

### 2. ป่า (Forest)
- หายาสมุนไพร
- มีวิญญาณหลอกหลอน
- มีพื้นที่น้ำเสีย/ของเสียอันตราย

### 3. วัด (Temple)
- สวดมนต์เพื่อฟื้นฟูจิตใจ
- รักษาโรคที่ติด
- ลดระดับภาพหลอน
- กด E ค้างไว้เพื่อสวดมนต์

## เงื่อนไขการชนะ

- รอดชีวิตผ่าน **7 วัน**
- รักษาผู้ป่วยครบตามจำนวนที่กำหนดในแต่ละวัน
- รักษาสุขภาพจิตไว้ได้
- ค้นพบยารักษาโรคระบาด

## เงื่อนไขการแพ้

- สุขภาพจิตหมด (Mental Health ≤ 0%)
- ภาพหลอนเต็มที่ (Hallucination ≥ 100%)

## การสร้าง Sprites

เกมนี้ใช้ pixel art แนว top-down จะต้องสร้าง sprites ต่อไปนี้:

### Player Sprites (16x16 หรือ 32x32 pixels)
- walk_up, walk_down, walk_left, walk_right (animation)
- idle_up, idle_down, idle_left, idle_right

### Environment
- สมุนไพร 4 ชนิด (8x8 pixels)
- ต้นไม้, หิน, หญ้า (tiles)
- วัด (32x32 pixels)
- หมู่บ้าน (buildings)
- น้ำเสีย/ของเสีย (effect tiles)

### Characters
- ผู้ป่วย (16x16 pixels)
- วิญญาณ (16x16 pixels, โปร่งแสง)

## การติดตั้ง

1. ดาวน์โหลด Godot 4.x จาก https://godotengine.org/
2. เปิดโปรเจคด้วย Godot Engine
3. สร้าง sprites และใส่ในโฟลเดอร์ assets/sprites/
4. ตั้งค่า Input Map ตามไฟล์ input_map.txt
5. สร้าง scenes ตามโครงสร้าง
6. กด F5 เพื่อรันเกม

## หมายเหตุสำหรับผู้พัฒนา

### ต้องสร้าง Scene เพิ่มเติม
1. **main_menu.tscn**: ปุ่มเริ่มเกม, ออกเกม
2. **intro.tscn**: แสดงเรื่องราว, ปุ่มดำเนินการต่อ
3. **main_game.tscn**: 
   - Player node
   - TileMap สำหรับแผนที่
   - GameUI (CanvasLayer)
   - PatientSpawner
   - Herb nodes
   - Spirit nodes
   - Temple node
   - HazardArea nodes
4. **game_over.tscn**: แสดงสถิติ, ปุ่มเล่นใหม่
5. **victory.tscn**: แสดงความสำเร็จ, ปุ่มกลับเมนู

### การตั้งค่า Autoload
เพิ่ม GameManager เป็น Autoload:
1. Project → Project Settings → Autoload
2. เพิ่ม scripts/game_manager.gd
3. ตั้งชื่อเป็น "GameManager"

### Tips
- ใช้ Collision Layers/Masks อย่างเหมาะสม
- เพิ่ม sound effects และ background music
- ปรับ balance ของเกมให้เหมาะสม
- เพิ่ม particle effects สำหรับความน่าสนใจ
